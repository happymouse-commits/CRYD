> 2026-06-12

---

## 三选一：你选路线，我就开工

### 🥇 路线A：全身立绘 + CSS 骨骼动画（最快，今晚出效果）

**怎么做**：
1. 用 SD img2img / ControlNet 从现有头像外扩生成**全身站立图**（写实风格，老师装扮）
2. 把身体拆成几个图层 PNG：头、躯干+手臂、下半身
3. 用 CSS `@keyframes` 做：
   - idle：呼吸起伏 + 头发微飘
   - speaking：嘴巴张合 + 手势微动
   - listening：身体前倾 + 眨眼
   - thinking：歪头思考 + 手指点下巴

**产出**：一个会呼吸、会眨眼、会做手势的全身虚拟老师
**时间**：今晚能出第一版
**依赖**：SD 出图（需要 GPU，你本地 RTX 5060 能用）

---

### 🥈 路线B：Rive 动画角色（中等成本，效果专业）

**怎么做**：
1. 用 SD 生成全身角色图
2. 导入 Rive（rive.app，免费）做骨骼绑定 + 动画
3. 前端用 `@rive-app/canvas` 加载 `.riv` 文件
4. 通过 API 控制动画状态切换

**产出**：丝滑的矢量/位图角色动画
**时间**：1-2 天
**优点**：动画质量高、性能好、文件小
**缺点**：需要学 Rive 编辑器

---

### 🥉 路线C：外包 Live2D（拉满效果）

**怎么做**：
1. 找画师画分层 PSD（全身、老师装扮）
2. 找 Live2D 建模师绑骨骼 + 做动画
3. 前端用 `pixi-live2d-display` 加载

**产出**：专业级可动角色
**时间**：外包周期 1-2 周
**成本**：500-2000 元（画师+建模师）
**优点**：效果拉满
**缺点**：花钱、周期长
