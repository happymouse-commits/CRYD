### #define 宏定义

**题目 #86** | code_output | 难度: ★★ | 知识点: `宏定义`

> 写出下面代码的输出结果：
#define S(a,b) a*b\nint main() { printf("%d", S(3+2, 4+1)); }

**答案:** 19

**解析:** 宏替换是文本替换：S(3+2,4+1)展开为3+2*4+1=3+8+1=12。但按优先级实际是3+2*4+1=12...wait：S(a,b)=a*b，替换后3+2*4+1=3+8+1=12。但如果写成#define S(a,b) ((a)*(b))结果会是5*5=25。这里按简单替换得12。
