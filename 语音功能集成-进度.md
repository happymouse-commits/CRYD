## 待完成：声音克隆

### 需要你提供

- **一段 10-30 秒的录音文件**（WAV/MP3/M4A/WebM 都行）
- 随便说什么内容，用于克隆你的声纹

### 操作步骤（拿到录音后）

1. 调用 MiMo 声音克隆 API 上传音频 → 获取 `voice_id`
2. 写入 `application.yml` 的 `mimo.api.voice-id` 字段
3. 重启后端，所有学生端 AI 回复自动用你的声音朗读

### 管理员录音页面

访问 `http://localhost:5173/voice-setup.html`（开发环境）或部署后访问同路径，按住按钮录音即可自动克隆。

## 新 API 接口汇总

| 方法 | 路径 | 用途 |
|------|------|------|
| POST | `/api/chat/voice-to-text` | 上传音频 → 返回文字（讯飞） |
| POST | `/api/chat/tts` | `{text}` → 返回 MP3 音频流（MiMo） |
| POST | `/api/chat/voice-clone` | 上传音频样本 → 克隆声音并持久化 |
| GET | `/api/chat/voice-id` | 查看当前已克隆的声音 ID |
| POST | `/api/chat/voice-id` | `{voiceId}` 手动设置声音 ID |

## 项目当前声音配置

```yaml
# application.yml
mimo:
  api:
    voice-id:   # ← 留空，等克隆后填入
```

克隆前默认使用 MiMo 内置女声 `default_zh`，填入 `voice-id` 后自动切换为克隆声音。
