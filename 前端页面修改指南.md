## 四、ECharts 图表配色

项目中使用了 ECharts（雷达图、折线图、树状图），**全部替换旧色**：

| 元素 | 旧值 | 新值 |
|------|------|------|
| 线条/点 | `#409EFF` | `#5b8def` |
| 面积填充 | `rgba(64,158,255,0.1)` | `rgba(91,141,239,0.1)` |
| 渐变起 | `rgba(64,158,255,0.35)` | `rgba(91,141,239,0.3)` |
| 渐变止 | `rgba(64,158,255,0.08)` | `rgba(91,141,239,0.06)` |

```javascript
// ✅ 正确
lineStyle: { color: '#5b8def', width: 2 },
areaStyle: { color: 'rgba(91,141,239,0.1)' },
itemStyle: { color: '#5b8def' }

// ❌ 错误（旧色，禁止使用）
lineStyle: { color: '#409EFF', width: 2 },
```

## 六、页面结构规范

### 6.1 Vue 单文件组件结构

```vue
<template>
  <div class="page-name">
    <!-- 页面内容 -->
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'
// 逻辑
</script>

<style scoped>
.page-name { }
/* 所有样式用 scoped */
</style>
```

### 6.2 布局层 vs 内容页

- **布局.vue** — 侧边栏 + 顶部栏 + `<router-view>`，不要在里面写业务内容
- **首页.vue / AI辅导.vue 等** — 业务页面，由 `<router-view>` 渲染
- 侧边栏常展 140px，白色背景，蓝色渐变 logo

### 6.3 页面命名

- 文件名用中文：`AI辅导.vue`、`我的画像.vue`
- 一个页面一个 `.vue` 文件，组件拆分到 `components/` 目录

## 七、禁止事项

- ❌ 不要使用 `#409EFF`、`#67C23A`、`#E6A23C`、`#F56C6C` 等 Element Plus 默认色
- ❌ 不要使用 `box-shadow: 0 2px 8px` 硬阴影，用 `0 1px 6px rgba(0,0,0,0.04)`
- ❌ 不要使用 `border-radius: 4px` 小圆角，卡片至少 12px，推荐 14-16px
- ❌ 不要把 API Key 写到源码
- ❌ 不要往 C 盘写文件
- ❌ 不要修改已有的比赛材料文件（.docx / .pptx），除非用户明确要求
- ❌ 不要在网页预览 HTML 里用 `file://` 协议引本地资源

## 八、完整改页面示例

假设你要把「教师端/布置作业.vue」改成亮色主题：

### Step 1：复制模板 CSS

从「2.1 通用卡片」「2.2 统计指标卡」复制对应的 CSS class。

### Step 2：执行颜色替换

对着「五、CSS 修改清单」逐项搜索替换。

### Step 3：调整 Element Plus 组件

```css
/* el-card 统一覆盖 */
.page :deep(.el-card) { border-radius: 14px; border: 1px solid #eef0f4; box-shadow: 0 1px 6px rgba(0,0,0,0.04); }
.page :deep(.el-card__header) { background: #fafbfd; border-bottom: 1px solid #eef0f4; }
/* el-table 头部 */
.page :deep(.el-table th) { background: #fafbfd; color: #6b7280; border-color: #eef0f4; }
```

### Step 4：构建验证

```bash
cd frontend && npm run build
```

构建通过即为成功，不要改逻辑，只改 CSS。
